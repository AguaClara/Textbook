Welcome to the AguaClara Textbook!¶

This textbook is written and maintained in Github via Sphinx. It uses and refers to AguaClara code and functions in aide_design. Listed below are the versions of the programs we use:

Table 1 These are the software versions used to compile this textbook¶
Software version
Sphinx 1.7.5
aide_design 0.0.12
Anaconda 4.5.4
Python 3.6.5
